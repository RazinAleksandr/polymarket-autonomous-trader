    ```python
    #!/usr/bin/env python3
    """
    Lightweight heartbeat -- runs every ~10 min via cron (NO API or LLM calls).
    
    Reads local state (trading.db, state/reports/) and writes state/signal.json
    with boolean flags that run_cycle.sh (Phase 4) reads to decide whether
    to launch an expensive Claude trading session.
    
    Flags (per D-08, adapted to 5-phase trading cycle):
      scan_needed:    True if no cycle report in last 4 hours (Phase B trigger)
      resolve_needed: True if any position's end_date within 24h or no resolve in 1h (Phase A trigger)
      learn_needed:   True if calibration_records newer than last learn report (Phase E trigger)
    """
    
    import json
    import os
    import sys
    from datetime import datetime, timezone, timedelta
    from pathlib import Path
    
    # Allow running from project root or scripts/ directory
    PROJECT_ROOT = Path(__file__).resolve().parent.parent
    sys.path.insert(0, str(PROJECT_ROOT))
    
    from lib.db import DataStore
    from lib.config import load_config
    ```

    **Constants:**
    ```python
    SIGNAL_FILE = PROJECT_ROOT / "state" / "signal.json"
    REPORTS_DIR = PROJECT_ROOT / "state" / "reports"
    SCAN_INTERVAL = timedelta(hours=4)     # Trigger scan every 4 hours
    RESOLVE_INTERVAL = timedelta(hours=1)  # Check resolves every 1 hour
    EXPIRY_WINDOW = timedelta(hours=24)    # Flag positions expiring within 24h
    EXPIRY_SOON_WINDOW = timedelta(hours=48)  # List positions expiring within 48h
    ```

    **Function: `latest_report_time(prefix: str, reports_dir: Path = REPORTS_DIR) -> datetime | None`**
    - Glob for `{prefix}*.md` in reports_dir
    - If no files: return None
    - Return mtime of most recent file as UTC datetime

    **Function: `scan_needed(now: datetime, reports_dir: Path = REPORTS_DIR) -> bool`**
    - t = latest_report_time("cycle-", reports_dir)
    - If None: return True (first run)
    - Return (now - t) > SCAN_INTERVAL

    **Function: `resolve_needed(now: datetime, store: DataStore, reports_dir: Path = REPORTS_DIR) -> bool`**
    - Check market_snapshots for open positions with end_date within EXPIRY_WINDOW:
      ```python
      open_positions = store.get_open_positions()
      open_market_ids = [p["market_id"] for p in open_positions]
      if open_market_ids:
          placeholders = ",".join("?" * len(open_market_ids))
          rows = store.conn.execute(
              f"SELECT DISTINCT market_id, end_date FROM market_snapshots "
              f"WHERE market_id IN ({placeholders}) AND end_date != '' "
              f"ORDER BY timestamp DESC",
              open_market_ids
          ).fetchall()
          for row in rows:
              try:
                  end_dt = datetime.fromisoformat(row["end_date"]).replace(tzinfo=timezone.utc)
                  if end_dt - now < EXPIRY_WINDOW:
                      return True
              except (ValueError, TypeError):
                  pass
      ```
    - Fallback: check last resolve report time
      - t = latest_report_time("resolve-", reports_dir)
      - If None: return True
      - Return (now - t) > RESOLVE_INTERVAL

    **Function: `learn_needed(now: datetime, store: DataStore, reports_dir: Path = REPORTS_DIR) -> bool`**
    - Per D-09: check calibration_records table for entries newer than last learn report
    - t = latest_report_time("learn-", reports_dir)
    - If t is None, check if any calibration_records exist:
      ```python
      row = store.conn.execute("SELECT COUNT(*) as cnt FROM calibration_records").fetchone()
      return row["cnt"] > 0
      ```
    - If t exists:
      ```python
      cutoff = t.isoformat()
      row = store.conn.execute(
          "SELECT COUNT(*) as cnt FROM calibration_records WHERE timestamp > ?",
          (cutoff,)
      ).fetchone()
      return row["cnt"] > 0
      ```

    **Function: `expiring_soon(now: datetime, store: DataStore) -> list[str]`**
    - Similar to resolve_needed but uses EXPIRY_SOON_WINDOW (48h)
    - Return list of question strings for positions approaching expiry

    **Function: `main(db_path: str = None, signal_path: Path = None, reports_dir: Path = None) -> dict`**
    - Parameters allow test injection (defaults use constants)
    - If db_path is None: config = load_config(); db_path = config.db_path
    - If signal_path is None: signal_path = SIGNAL_FILE
    - If reports_dir is None: reports_dir = REPORTS_DIR
    - now = datetime.now(timezone.utc)
    - store = DataStore(db_path)
    - Build signal dict:
      ```python
      signal = {
          "generated_at": now.isoformat(),
          "scan_needed": scan_needed(now, reports_dir),
          "resolve_needed": resolve_needed(now, store, reports_dir),
          "learn_needed": learn_needed(now, store, reports_dir),
          "expiring_soon": expiring_soon(now, store),
          "open_positions": len(store.get_open_positions()),
      }
      ```
    - store.close()
    - Ensure parent dir exists: signal_path.parent.mkdir(parents=True, exist_ok=True)
    - Write signal.json with indent=2
    - Print summary to stdout (like reference implementation)
    - Return signal dict (for testing)
