    import os

    **Function: `_fetch_etf_sma(symbol: str, api_key: str, period: int = 20) -> dict | None`**
    - GET Alpha Vantage TIME_SERIES_DAILY for symbol, outputsize=compact
    - Extract last `period` closing prices from "Time Series (Daily)"
    - Compute SMA = mean of last `period` closes
    - current_price = most recent close
    - Return {"symbol": symbol, "current": current_price, "sma_20": sma, "above_sma": current_price > sma}
    - On failure: log.error, return None

    **Function: `get_macro_regime(api_key: str | None = None) -> dict`**
    - If api_key is None: api_key = os.environ.get("ALPHA_VANTAGE_API_KEY")
    - If still None: return {"regime": None, "etfs": {}, "signals": [], "warnings": ["ALPHA_VANTAGE_API_KEY not set"]}
    - Fetch SMA data for QQQ, XLP, GLD, UUP via _fetch_etf_sma
    - Classification (per D-05):
      - risk_on_signals: QQQ above SMA, GLD below SMA
      - risk_off_signals: QQQ below SMA, GLD above SMA
      - Count signals: if risk_on >= 2 and risk_off == 0: "risk-on"
      - If risk_off >= 2 and risk_on == 0: "risk-off"
      - Else: "mixed"
    - Build signals list: e.g. ["QQQ above 20d SMA (bullish)", "GLD below 20d SMA (risk-on)"]
    - Return {"regime": str, "etfs": {symbol: {current, sma_20, above_sma}}, "signals": list, "warnings": list}
    - Partial failures: include available ETFs, add warning for each failed fetch

    **Function: `get_fear_greed() -> dict`**
    - GET https://api.alternative.me/fng/?limit=1 with timeout=10
    - Parse response: data[0].value (int), data[0].value_classification (str)
    - Return {"value": int, "classification": str}
    - On failure: return {"value": None, "classification": None, "warnings": ["Fear & Greed API error: {e}"]}

    **Function: `get_category_news(category: str, api_key: str | None = None, limit: int = 5) -> dict`**
    - Per D-07: Use Alpha Vantage NEWS_SENTIMENT endpoint
    - If no api_key: return {"articles": [], "warnings": ["ALPHA_VANTAGE_API_KEY not set"]}
    - Map category to topics/tickers: crypto -> "CRYPTO", politics -> "POLITICS", sports -> "SPORTS", etc.
    - GET ALPHA_VANTAGE_BASE with function=NEWS_SENTIMENT, topics=mapped_topic, limit=limit, sort=LATEST
    - Parse feed: extract title, url, overall_sentiment_score, overall_sentiment_label, time_published
    - Return {"articles": list[dict], "avg_sentiment": float, "warnings": list}
    - On failure: return {"articles": [], "avg_sentiment": None, "warnings": [str(e)]}

    **Function: `get_category_intel(category: str, api_key: str | None = None) -> dict`**
    - Combines macro + fear_greed (if crypto) + category news
    - macro = get_macro_regime(api_key)
    - fear_greed = get_fear_greed() if category == "crypto" else None
    - news = get_category_news(category, api_key)
    - Aggregate warnings from all sources
    - Return:
    ```python
    {
        "category": category,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "macro_regime": macro["regime"],
        "macro_signals": macro["signals"],
        "fear_greed": fear_greed,
        "news": news["articles"],
        "avg_news_sentiment": news["avg_sentiment"],
        "warnings": all_warnings
    }
    ```

    **Function: `get_market_overview(api_key: str | None = None) -> dict`**
    - macro = get_macro_regime(api_key)
    - fear_greed = get_fear_greed()
    - Return:

        "categories": CATEGORIES,
        "warnings": macro.get("warnings", []) + fear_greed.get("warnings", [])
    }
