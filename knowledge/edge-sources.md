## How to Update This File

When a trade resolves as a WIN:
1. Find the relevant edge source in Testing or Confirmed
2. Increment confirmed_wins
3. Recalculate success_rate = wins / (wins + losses)
4. Add a brief note about what confirmed the edge
5. If success_rate > 50% with 3+ wins: promote to Confirmed Edge Sources

When a trade resolves as a LOSS:
1. Find the relevant edge source
2. Increment losses
3. Add note about what failed
4. If success_rate < 30% with >= 5 trades: move to Failed Edge Sources

When discovering a new potential edge:
1. Add to Hypothesized Edge Sources with a brief hypothesis
2. Move to Testing Edge Sources only after finding 1+ actionable example with an open trade
